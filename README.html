# SMBC 自动记账系统 · smbc-auto-ledger > 基于 Google Apps Script + Google Sheets 的个人消费自动化记账系统。 > 从 SMBC 银行邮件自动解析交易、自动分类、写入 Sheet，并定期发送消费周报 / 月报。 --- ## 目录 - [系统概览](#系统概览) - [数据流](#数据流) - [Sheet 结构](#sheet-结构) - [核心功能](#核心功能) - [函数入口速查](#函数入口速查) - [配置常量](#配置常量) - [报告格式](#报告格式) - [使用方法](#使用方法) - [修复记录与踩坑备忘](./CHANGELOG.html) --- ## 系统概览 一封 SMBC 邮件到账 → Gmail 抓取 → 正则解析 → 分类规则引擎 → 写入 Sheet → 聚合成周/月报 → 定时发送。全自动，无人工介入。 支持三种邮件格式：**刷卡通知 · 银行出金通知 · 口座引き落とし事前お知らせ**。 --- ## 数据流 ``` SMBC 邮件 → Gmail → GAS 解析 → 取引 Sheet → 日次 Sheet（汇总） → 邮件报告（周 / 月） ``` --- ## Sheet 结构 ### 「取引」— 主流水表 | 列 | 字段 | 说明 | ### 按月自动轮换 6 套主题

同一双月的周报和月报使用同一主题，一年正好完整循环一轮：

| 月份 | 模板 ID | 风格 |
|---|---|---|
| 1–2 月   | `c6` | 八十年代账本（红格线 + 仿宋，年关清账感） |
| 3–4 月   | `c4` | 古诗主题（楷体 + 宣纸，月份对应应景诗句） |
| 5–6 月   | `c1` | 自然现代宋体（米色新闻纸，明快朴素） |
| 7–8 月   | `a`  | 纯净英文新闻纸（白底 Georgia，WSJ 风） |
| 9–10 月  | `c2` | 中文报纸社论（粗宋体 + 红索引条） |
| 11–12 月 | `e7` | 维多利亚粗 slab（Playfair + 羊皮色） |

调度逻辑（`emailTemplates.gs` 的 `pickTemplateId_`）：

```javascript
const EMAIL_TEMPLATES = ['c6', 'c4', 'c1', 'a', 'c2', 'e7'];
// new Date().getMonth() 返回 0–11，除以 2 向下取整 → 0–5
EMAIL_TEMPLATES[Math.floor(date.getMonth() / 2)];
```

### 周报 vs 月报

共享同一模板，内部通过 `includeFixed` 参数区分：

- **月报**（每月 1 号）：渲染全部 6 个区块 — 报头、Lead、TOTAL/DAILY、分类明细、**固定估算**、**Grand Total**、签名、纯文本复制区
- **周报**（每周一）：裁掉固定估算和 Grand Total 两块（一周无意义叠估算），其它完全一致
